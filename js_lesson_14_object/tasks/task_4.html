Задача 4.З використанням замикань розробити ітератор, тобто функцію, що буде поступово за окремими викликами видавати по одній букві наперед заданого слова. З використанням нього розробити гру «Прекладач». Виводимо для користувача опис цього слова і по одну букву користувач вводить переклад цього слова. Підрахувати кількість вгаданих букв.
